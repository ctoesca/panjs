Container component

Create the file Tpanel.html in "components" directory

Put the component in app.html

Modify app.html <body>

<body> <p><code>app.html</code> component</p> <div data-compo="components.Tpanel.html"> <h4>Normal DOM element</h4> <div data-compo="components.TtestButton.html"/> <div data-compo="components.TtestButtonExt.html"/> </div> </body>

Notes:

  • Tpanel is like another component except that we have added <!--CONTENT--> in the body.
  • When we use Tpanel, we can put others components or DOM elements in the Tpanel element. <!--CONTENT--> is replaced by these elements.
