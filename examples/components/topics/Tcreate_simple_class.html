Create a simple class

We have seen how to create HTML components but you can also create pure javascript classes using defineClass().
The same principle is applicable, except that you can't put CSS or HTML in the javascript file.

Example : the core.helpers.Ttimer.js class

This class is a timer. When start() is called, it that send a Ttimer.ON_TIMER each delay seconds.

Instantiating the class

uses("core.helpers.Ttimer"); var timer = new Ttimer({delay: 1000});
