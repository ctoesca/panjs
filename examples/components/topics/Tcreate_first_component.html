Creating a simple component

Create the file TtestButton.html in "components" directory

HTML components must be well-formed XML.

Notes:

  • this.btnSubmit and this.result are jQuery objects. You can access them like this because they have id="btnSubmit" and id="result".
  • The component container is a div because we have set the property baseElement="div"

Put the component in app.html

Modify app.html <body>

<body> <p><code>app.html</code> component</p> <div data-compo="components.TtestButton.html"/> </body>
