Pass arguments

Passing arguments in the component HTML markup

You can pass argument directly in the HTML markup, adding attributes prefixed by data-.
Attributes without -data are allowed but we recommend to use -data.
Use small letters for attributes names.

Example :

<body> <div data-max-clicks="3" data-compo="components.TtestButton.html"></div> </body>

Getting arguments in the component :

Arguments are available in the args object, in the contructor.

<script subtype="text/x-class-definition"> //<![CDATA[ defineClass("TtestButton", "core.display.Telement", { baseElement: "div", constructor: function(args){ this._super.constructor.call(this,args); alert( "maxClicks = "+ args.maxClicks ); } }); //]]> </script>

In the args object, the arguments names are not the same as the attributes names :

  • data- is removed
  • The name always starts with a lower case letter
  • Letters after each - are upper case letters

Exemples:

  • data-max-clicks will be renamed to maxClicks
  • data-maxclicks will be renamed to maxclicks
  • data-maxClicks will be renamed to maxclicks

Passing arguments when you call load()

When you load a component lazily, you can pass arguments:

load({args1: "bla", arg2: 3})

Passing arguments when you create a component programmatically

var compo = panjs.createComponent("components.TtestButton.html",{args1: "bla", arg2: 3});
