Extend a container component

Create the file TpanelExt.html in "components" directory

Notes:

  • The <body> of this component is injected in the <!--CONTENT--> of the parent.
  • Like each container component, it contains a <!--CONTENT--> element.

Modify app.html

In app.html <body>, replace Tpanel by TpanelExt

<body> <p><code>app.html</code> component</p> <div data-compo="components.TpanelExt.html"> <h4>Normal DOM element</h4> <div data-compo="components.TtestButton.html"/> <div data-compo="components.TtestButtonExt.html"/> </div> </body>
