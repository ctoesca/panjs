Creating a simple component

Create the file TtestButton.html in "components" directory

HTML components must be well-formed XML.

Notes:

  • The DOM element of the component will be a div element because we set baseElement: "div" in the component properties.
  • this.btnSubmit and this.result are jQuery objects. You can access them like this because the framework maps HTML elements id="btnSubmit" and id="result" on this object.

Put the component in app.html

Modify app.html <body>

<body> <p><code>app.html</code> component</p> <div data-compo="components.TtestButton.html"/> </body>
