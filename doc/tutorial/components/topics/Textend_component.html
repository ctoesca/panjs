Extend a component

Create the file TtestButtonExt.html in "components" directory

Notes:

  • In the defineClass function, the component inherits from components.TtestButton.html
  • We override the style using TtestButtonExt class selector
  • We override onSuccess method and call the parent doing: this._super.onSuccess.call(this);
  • We access this.btnSubmit defined in the parent component.
  • In this example, we don't override the body html

Put the component in app.html

Modify app.html <body>

<body> <p><code>app.html</code> component</p> <div data-compo="components.TtestButton.html"/> <div data-compo="components.TtestButtonExt.html"/> </body>
