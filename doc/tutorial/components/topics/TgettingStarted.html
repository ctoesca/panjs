Getting started

Download

Donwload source code on gitHub

Create your application directory structure 

    <root directory>
        |--panjs
        |--myApplication
            |--css
            |--components
              |--app.html
            |--config.html
            |--index.html

Create index.html and app.html

index.html is the main page. panjs_core.min.js must be loaded after var panjs = { ... }.
app.html is the main component: it contains others components.
For convenience, we use bootstrap style sheet and FontAwesome but it's optional.

<!DOCTYPE html> <html lang="en"> <head> <script src="//ajax.googleapis.com/ajax/libs/jquery/2.1.1/jquery.min.js"></script> <!--Bootstrap style (optional) --> <link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/bootstrap/3.3.1/css/bootstrap.min.css"/> <!--Font Awesome (optional) --> <link rel="stylesheet" href="//maxcdn.bootstrapcdn.com/font-awesome/4.3.0/css/font-awesome.min.css"/> <script> //<![CDATA[ var panjs = { logLevel: "DEBUG", env: "dev", // dev or prod appVersion: "1.0.0", version: "0.7.4", //panjs version namespaces:{ "core": {path: "../panjs/core"}, "ui": {path: "../panjs/ui"}, "components": {path: "./components"} } }; //]]> </script> <script src="../panjs/core/panjs_core_with_less.dev.min.js"></script> <script> $(document).ready(function() { panjs.load($(document.body)) ; }); </script> </head> <body> <div data-compo="components.app.html"/> </body> </html>
